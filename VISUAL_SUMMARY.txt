│  📡 API ENDPOINTS (20+)                                              │
└──────────────────────────────────────────────────────────────────────┘

🌐 WEB INTERFACE
   GET  /                          Ana sayfa (Dashboard)
   GET  /styles.css                CSS dosyası
   GET  /app.js                    JavaScript dosyası

💬 CHAT
   GET  /api/chat/messages         Mesajları listele
   POST /api/chat/messages         Mesaj gönder

🎤 VOICE CALLS
   POST /api/voice/call            Arama başlat
   POST /api/voice/answer/:id      Aramayı yanıtla
   POST /api/voice/end/:id         Aramayı bitir
   GET  /api/voice/active          Aktif aramalar

📹 VIDEO CALLS
   POST /api/video/call            Video arama başlat
   POST /api/video/answer/:id      Aramayı yanıtla
   POST /api/video/ice/:id         ICE candidate ekle
   POST /api/video/end/:id         Aramayı bitir
   GET  /api/video/active          Aktif video aramalar

📺 LIVE STREAMING
   POST /api/stream/start          Yayın başlat
   GET  /api/stream/live           Canlı yayınlar
   POST /api/stream/join/:id       Yayına katıl
   POST /api/stream/leave/:id      Yayından ayrıl
   POST /api/stream/end/:id        Yayını bitir
   GET  /api/stream/:id            Yayın detayları

🏥 UTILITY
   GET  /api/health                Sunucu durumu
   GET  /api                       API dokümantasyonu

# TARAYICIDA AÇ
http://localhost:8080

# CHAT'TE MESAJ GÖNDER
1. Kullanıcı adını gir (sağ üstte)
2. Sohbet sekmesine git
3. Mesajını yaz
4. Enter'a bas veya Gönder'e tıkla

# VİDEO ARAMA BAŞLAT
1. Görüntülü Arama sekmesine git
2. Alıcı bilgilerini gir
3. "Video Aramayı Başlat" butonuna tıkla
4. Aktif aramalardan takip et

# CANLI YAYIN BAŞLAT
1. Canlı Yayın sekmesine git
2. Başlık ve açıklama gir
3. "Yayını Başlat" butonuna tıkla
4. Stream key'ini al
5. Canlı yayınlar bölümünden izle
